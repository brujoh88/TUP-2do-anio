### Exercise 3 (p.23) - Complete dialogue with Present Continuous

> **A:** Hi, how is the project going? What **are** you **doing** now?
>
> **B:** I **am working** on the homepage layout. I **am using** HTML and CSS at the moment.
>
> **A:** OK. **Are** you **working** on the JavaScript file too?
>
> **B:** No, I **am not**. I usually do that after finishing the layout.
>
> **A:** That makes sense. Yesterday, I finished the button design and tested it in the browser.
>
> **B:** Great. Right now, Lisa and I **are adding** the navigation menu. It **is not working** correctly on mobiles yet.
>
> **A:** **Is** Lisa **fixing** it now, or **is** she still **testing** it?
>
> **B:** I **am fixing** the CSS now, but she cannot test it yet.
>
> **A:** OK. Let me know when you finish. We always test the website before publishing it.

### Step 2: Use the Prompts (p.26-27) - Zillow website

This landing page **is promoting** an online real estate service. (promote - present continuous)

The header is **at the top of** the page. (preposition of place)

The logo is **next to** the navigation menu, and the menu is at **the top** of the page.

The main title is **in** the center of the page. (preposition of place)

Below the title, the website **is showing** a search bar for users. (show - present continuous)

The search button is **next to** the input field. (preposition of place)

The page **is inviting** users to search for homes and agents. (invite - present continuous)

**Enter** the address in the search bar and **click** the search button. (imperatives)

**Do not** forget to check the results after searching. (negative imperative)
