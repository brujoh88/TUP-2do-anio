## 1. Vocabulario de partes de un sitio web

| Ingles | Español | Que es |
|--------|---------|--------|
| Header | Encabezado | Parte superior del sitio, contiene logo y menu de navegacion |
| Navigation Bar (Navbar) | Barra de navegacion | Menu con links para moverse por el sitio |
| Footer | Pie de pagina | Parte inferior, tiene info de contacto o links |
| Homepage | Pagina principal | Primera pagina que ves al entrar al sitio |
| Button | Boton | Elemento clickeable que ejecuta una accion |

| Icon | Icono | Imagen pequeña que representa algo (ej: lupa = buscar) |
| Banner | Banner | Imagen ancha o seccion en la parte superior de la pagina |

## 4. Pasos del desarrollo web (sequence words)

El booklet enseña a describir procesos usando **palabras de secuencia**:

| Ingles | Español | Uso |
|--------|---------|-----|
| First | Primero | Primer paso |
| Secondly | En segundo lugar | Segundo paso |
| Next | Luego / Despues | Siguiente paso |
| Then | Entonces / Luego | Siguiente paso |
| After that | Despues de eso | Siguiente paso |
| Finally | Finalmente | Ultimo paso |

**Pasos del desarrollo web en orden:**
1. **First**, discuss with the customer their requirements → Primero, discutir con el cliente sus requisitos
2. **Secondly**, analyse the information → Segundo, analizar la informacion
3. **Then**, create a website specification → Luego, crear una especificacion del sitio
4. **After that**, design and develop the website → Despues, diseñar y desarrollar el sitio
5. **Next**, assign a specialist to write content → Luego, asignar un especialista para escribir contenido
6. **After that**, give the project to programmers for HTML coding → Dar el proyecto a programadores para codificar en HTML
7. **Finally**, test the website → Finalmente, testear el sitio

## 5. Preposiciones de lugar (Prepositions of Place)

Se usan para describir **donde estan los elementos** en una pagina web:

| Preposicion | Significado | Ejemplo |
|-------------|------------|---------|
| **at the top (of)** | En la parte superior | The logo is **at the top** of the page → El logo esta en la parte superior |
| **at the bottom (of)** | En la parte inferior | The footer is **at the bottom** → El pie de pagina esta abajo |
| **below / under** | Debajo de | The banner is **below** the header → El banner esta debajo del encabezado |
| **in** | Dentro de | The menu is **in** the header → El menu esta dentro del encabezado |
| **on** | Sobre (en una superficie/pagina) | There is a form **on** the homepage → Hay un formulario en la pagina principal |
| **next to** | Al lado de | The search bar is **next to** the logo → La barra de busqueda esta al lado del logo |
| **on the right/left** | A la derecha/izquierda | The search bar is **on the right** → La barra de busqueda esta a la derecha |

## 6. Imperativos (Imperatives)

Se usan para dar instrucciones. Muy comunes en programacion y documentacion tecnica.

**Estructura:**
- **Afirmativo:** Verbo base + objeto → `Open your code editor` (Abri tu editor de codigo)
- **Negativo:** Do not/Don't + verbo base + objeto → `Do not change the code` (No cambies el codigo)

**No llevan sujeto** (no se dice "You open...", solo "Open...").

**Ejemplos del booklet:**
- **Add** a main title using `<h1>` → Agrega un titulo principal usando `<h1>`
- **Include** the basic structure: `<html>`, `<head>`, `<body>` → Incluí la estructura basica
- **Open** your code editor → Abrí tu editor de codigo
- **Do not** change the code today → No cambies el codigo hoy

## 7. Presente Continuo (Present Continuous)

Se usa para hablar de acciones que estan pasando **ahora mismo** o **en este periodo**.

| Forma | Estructura | Ejemplo | Traduccion |
|-------|-----------|---------|-----------|
| **Afirmativa** | Sujeto + am/is/are + verbo-ing | I **am working** on a project | Estoy trabajando en un proyecto |
| **Negativa** | Sujeto + am/is/are + NOT + verbo-ing | The website **is not working** | El sitio no esta funcionando |
| **Pregunta** | Am/Is/Are + sujeto + verbo-ing? | **Are** you **working** on the website? | ¿Estas trabajando en el sitio? |

**Expresiones de tiempo comunes:**
- Now / Right now → Ahora / Justo ahora
- At the moment / At this moment → En este momento
- At present → Actualmente
- Today → Hoy

**Ejemplo del texto:** "Now, she **is adding** new features to make it better" → Ahora, ella esta agregando nuevas funcionalidades para mejorarlo.

## 8. Texto clave: "A web developer's work" (pag 6)

Resumen del caso de estudio de Andrea:
- Andrea es **web developer** (desarrolladora web)
- Trabajo en un sitio para **SLFan** (fan club del juego Space Liberator)
- Tecnologias usadas: **PHP** (reduce costos), **HTML5** (bueno para videos e imagenes), **MySQL** (guarda datos privados como posts del foro)
- Testeo el sitio en **celulares y tablets**
- Estructura del sitio: **logo en el centro arriba**, **menu arriba**, **barra de busqueda a la derecha**, debajo del banner, **contacto abajo** del todo
