### Exercise 3 - Karl's career: Past Simple + Modal 'Can' (p.19)

| # | Verbo base | Forma correcta | Tipo |
|---|-----------|---------------|------|
| 1 | study | **studied** | Past Simple regular (y → ied) |
| 2 | leave | **left** | Past Simple irregular |
| 3 | go | **went** | Past Simple irregular |
| 4 | graduate | **graduated** | Past Simple regular |
| 5 | decide | **decided** | Past Simple regular |
| 6 | get | **got** | Past Simple irregular |
| 7 | stay | **stayed** | Past Simple regular |
| 8 | go | **went** | Past Simple irregular |
| 9 | can/can't | **can't** | Modal verb (negativo) |
| 10 | want | **wanted** | Past Simple regular |

**Texto completo:**

> I **studied** and **left** school in Cambridge in the UK at the age of 18 and **went** to the University of Bristol to study computing in 2020. I **graduated** in 2024 and **decided** to travel around the world for a year.
> In 2025, I **got** a job back in Cambridge with the software company Arm. I **stayed** with Arm for two years and **went** to work for Microsoft in Seattle in the USA because there I **can't** develop as a web page designer - this is what I **wanted** to be. I am a well known page designer now and I love it.

**Explicacion:** El texto usa Past Simple para narrar la carrera de Karl en orden cronologico. El "can't" indica que en Arm no podia desarrollarse como web page designer, por eso se fue a Microsoft. "wanted" esta en pasado porque describe lo que queria ser en ese momento.

# 05 Written Production

### Step 2 - Complete the paragraph using words from the box (p.22)

Word bank: selectors, styles, HTML, allows, colours, tags, pages, rule, CSS

> CSS (Cascading Style Sheets) **allows** you to apply styles to HTML documents by prescribing **colours**, fonts, spacing, and positioning. The main advantages are the separation of content (in **HTML**) and styling (in **CSS**) and the same CSS rules can be used across all **pages** and not have to be rewritten. HTML uses **tags**, and CSS uses **rule** sets. CSS styles are applied to the HTML element using **selectors**.

**Explicacion:** CSS permite (allows) aplicar estilos prescribiendo colores (colours). La separacion de contenido (HTML) y estilo (CSS) es la ventaja principal. HTML usa tags, CSS usa rule sets, y los estilos se aplican mediante selectors.

### Step 3 - Write a short technical report (80-100 words) (p.22-23)

> CSS is a styling language designed to simplify the process of making web pages presentable. It allows developers to control colours, fonts, and layout independently from HTML. One important advantage is that CSS can be applied across multiple pages, so developers do not need to repeat code. This is faster and easier to maintain than writing inline styles. However, one disadvantage is that different browsers can display CSS differently, so developers need to test their code on multiple platforms. A light background is better than a dark one because it is easier to read long texts. In conclusion, CSS is one of the best tools for creating professional and responsive websites.
