## Contenido del booklet

### 01 - Reading Comprehension
- **Text 1 - Webpage Features** (p.3-5): Pre-reading, "Building Your First Website", True/False, completar
- **Text 2 - Webpage Design** (p.5-7): HTML vs CSS vs JS, "Promote your Business", ventajas/desventajas CSS, True/False

### 02 - Listening Comprehension
- **2.1 Webpage Description** (p.9-10): Entrevista a John sobre su website
- **2.2 Website Analytics** (p.11-12): Descripcion de web page, ordenar oraciones, comparativos

### 03 - Vocabulary & Grammar (p.13-19)
- Vocabulario web design (CSS, layout, font, button, responsive, domain, SEO, CMS, UI, UX, HTML, Navigation)
- Unscramble words (link, content, background)
- **Comparatives & Superlatives** (bigger, faster, easier, more/less + adj, the best, the worst)
- **Connectors** (because, so, so that, in order to)
- **Past Simple Tense** (regular: -ed, irregular: had/changed/went)
- **Modal verb 'Can'** (posibilidad, capacidad)
- Practice: comparativos, texto de Karl (past simple + can)

### 04 - Written Production (p.20-24)
- Step 1: Read the Model ("Building Your First Website")
- Step 2: Completar parrafo con palabras (selectors, styles, HTML, allows, etc.)
- Step 2.3: Escribir oraciones sobre CSS (comparative, can, connectors)
- Step 3: Escribir reporte tecnico propio (80-100 palabras)

### 05 - Speaking Production (p.25-28)
- Oral Presentation: Web page design usando CSS (3-4 minutos)
- Requisitos: 3+ web design words, 2 comparatives, 1 superlative, connectors
- Useful vocabulary para presentar (layout, font, colours, buttons, comparing designs)
- Self-assessment checklist
