## Resumen rapido

| Concepto | Clave |
|----------|-------|
| Link (href) | `<a href="url">` - href es el atributo para la URL de destino |
| Video controls | `<video controls>` - muestra controles de reproduccion |
| Autoplay | `<video autoplay>` - reproduce automaticamente al cargar |
| `<img>` vs `<video>` | `<img>` es vacia (sin cierre), `<video>` lleva cierre `</video>` |
| Atributo alt | Descripcion alternativa para accesibilidad y SEO |

## Preguntas

### Enlaces y multimedia

**1. En `<a>`, ¿que atributo especifica la URL de destino?**
**R: `href`**

**2. ¿Que atributo de `<video>` da controles de reproduccion?**
**R: `controls`**

**3. ¿Que atributo permite reproduccion automatica del video?**
**R: `autoplay`**

### Etiquetas y atributos

**4. ¿Cual afirmacion sobre cierre de etiquetas es correcta?**
**R: `<img>` es vacia y no lleva cierre, mientras que `<video>` si.** `<img>` es autoconclusiva, `<video>` necesita `</video>`.

**5. ¿Funcion principal del atributo `alt` en `<img>`?**
**R: Proporcionar una descripcion alternativa para accesibilidad y SEO.**
