| Concepto | Clave |

### Semantica HTML5

**2. ¿Diferencia entre `<section>` y `<article>`?**
**R: `<section>` agrupa contenido relacionado, `<article>` es contenido independiente** que podria distribuirse de forma aislada.

**3. ¿Que etiqueta agrupa contenido como unidad independiente?**
**R: `<article>`**

**4. ¿Que etiqueta es correcta para el contenido principal unico?**
**R: `<main>`**

**10. ¿Que elemento define el pie de pagina?**

### Formularios

**5. ¿Que atributo hace que un checkbox aparezca marcado por defecto?**
**R: `checked`**

### Listas

**6. ¿Que etiqueta se usa para listas ordenadas?**
**R: `<ol>`** (ordered list). `<ul>` es desordenada, `<li>` es cada item.

### Tablas

**7. ¿Que elemento representa una celda de encabezado?**
**R: `<th>`** (celda individual en negrita y centrada).

**9. ¿Que etiqueta se usa para la cabecera de una tabla?**
**R: `<thead>`** (seccion que contiene las filas de encabezado).
