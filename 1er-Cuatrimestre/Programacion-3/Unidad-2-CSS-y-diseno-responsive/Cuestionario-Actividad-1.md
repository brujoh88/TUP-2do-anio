# Cuestionario - Actividad 1: Fundamentos CSS

Resultado: **10/10** (100%)

## Resumen rápido

| Concepto | Clave |
|----------|-------|
| `color` | Cambia el color del texto de un elemento |
| `background-color` | Cambia el color de fondo de un elemento |
| Propósito de CSS | Separar el contenido (HTML) de la presentación visual (Diseño) |
| `.` (punto) | Prefijo para seleccionar una clase en CSS |
| `#` (numeral) | Prefijo para seleccionar un ID único en CSS |
| CSS | Cascading Style Sheets |
| `p` | Selector de tipo para aplicar estilo a todos los párrafos |
| `<link>` | Forma recomendada de incluir CSS (archivo externo) |
| `font-size` | Propiedad en `p { font-size: 16px; }` |
| `h1` | Selector en `h1 { color: red; }` |

---

## Preguntas y respuestas

**1. ¿Cuál es la propiedad correcta para cambiar el color del TEXTO de un elemento?**
- **Respuesta: `color`**

**2. ¿Cuál es la propiedad correcta para cambiar el color de FONDO de un elemento?**
- **Respuesta: `background-color`**

**3. ¿Cuál es el propósito principal de utilizar CSS en un sitio web?**
- **Respuesta: Separar el contenido (HTML) de la presentación visual (Diseño)**

**4. ¿Qué carácter se utiliza como prefijo para seleccionar una CLASE en CSS?**
- **Respuesta: `.` (punto)**

**5. ¿Qué carácter se utiliza como prefijo para seleccionar un ID único en CSS?**
- **Respuesta: `#` (numeral)**

**6. ¿Qué significan las siglas CSS?**
- **Respuesta: Cascading Style Sheets**

**7. Si quieres aplicar un estilo a TODOS los párrafos de tu sitio web, ¿qué selector deberías usar?**
- **Respuesta: `p`**

**8. ¿Cuál es la forma recomendada de incluir CSS en un proyecto profesional?**
- **Respuesta: En un archivo externo vinculado mediante la etiqueta `<link>`**

**9. En la regla CSS `p { font-size: 16px; }`, ¿cuál es la propiedad?**
- **Respuesta: `font-size`**

**10. En la regla CSS `h1 { color: red; }`, ¿cuál es el selector?**
- **Respuesta: `h1`**
