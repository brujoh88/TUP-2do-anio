# Cuestionario - Actividad 3: Flexbox y Media Queries

Resultado: **10/10** (100%)

## Resumen rápido

| Concepto | Clave |
|----------|-------|
| Breakpoint tablets | `768px` |
| `flex-grow` | Capacidad de un ítem para crecer y ocupar espacio sobrante |
| `@media` | Regla (at-rule) para definir una Media Query |
| `align-items` | Alinea elementos en el eje transversal (perpendicular al principal) |
| `and` | Operador lógico para combinar condiciones en media queries |
| `flex-direction` | Define la dirección del eje principal (main axis) |
| `prefers-reduced-motion` | Detecta si el usuario prefiere reducir animaciones |
| `justify-content` | Alinea elementos en el eje principal (horizontal si `flex-direction: row`) |
| Flexbox | Unidimensional, distribuye espacio dinámicamente entre elementos |
| Mobile First | Se usa `min-width` para aplicar estilos a medida que la pantalla crece |

---

## Preguntas y respuestas

**1. ¿Cuál es un breakpoint estándar comúnmente utilizado para tablets?**
- **Respuesta: `768px`**

**2. ¿Qué propiedad de un ítem flexible define su capacidad para crecer y ocupar el espacio sobrante?**
- **Respuesta: `flex-grow`**

**3. ¿Cuál es la regla correcta ("at-rule") para definir una Media Query en CSS?**
- **Respuesta: `@media`**

**4. ¿Qué propiedad se utiliza para alinear los elementos en el eje transversal (perpendicular al eje principal)?**
- **Respuesta: `align-items`**

**5. ¿Qué operador lógico se utiliza para combinar múltiples condiciones en una media query?**
- **Respuesta: `and`**

**6. ¿Qué propiedad define la dirección del eje principal (main axis) en un contenedor flexible?**
- **Respuesta: `flex-direction`**

**7. ¿Qué característica de media query permite detectar si el usuario prefiere reducir las animaciones?**
- **Respuesta: `prefers-reduced-motion`**

**8. Si `flex-direction` es `row`, ¿qué propiedad alinea los elementos horizontalmente (eje principal)?**
- **Respuesta: `justify-content`**

**9. ¿Cuál es la característica principal del modelo de diseño Flexbox?**
- **Respuesta: Es unidimensional y distribuye el espacio dinámicamente entre los elementos.**

**10. En el enfoque "Mobile First", ¿qué propiedad se utiliza principalmente para aplicar estilos a medida que la pantalla crece?**
- **Respuesta: `min-width` (ancho mínimo)**
