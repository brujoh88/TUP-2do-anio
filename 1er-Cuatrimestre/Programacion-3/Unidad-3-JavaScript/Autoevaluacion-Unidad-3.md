## Preguntas y respuestas

**1. ¿Qué método crea un nuevo array con los elementos que cumplen una condición?**
- **Respuesta: `filter()`**

**2. ¿Qué palabra clave se usa para declarar una constante?**
- **Respuesta: `const`**

**3. ¿Qué bucle garantiza al menos una ejecución aunque la condición sea falsa?**
- **Respuesta: `do while`**

**4. ¿Qué método del DOM devuelve un solo elemento que cumpla con un selector CSS?**
- **Respuesta: `querySelector()`**

**5. ¿Qué es JavaScript?**
- **Respuesta: Un lenguaje de programación interpretado**

**6. ¿Cuál es la salida de `5 === "5"`?**
- **Respuesta: `false`**

**7. ¿Qué operador lógico devuelve `true` si ambos valores son `true`?**
- **Respuesta: `&&`**

**8. ¿En qué entornos se puede ejecutar JavaScript?**
- **Respuesta: En navegadores y también en servidores**

**9. ¿Cuál es una de las principales razones por las que JavaScript es ampliamente utilizado en desarrollo web?**
- **Respuesta: Porque permite crear páginas web dinámicas e interactivas**

**10. ¿Cuál es la diferencia entre `let` y `var`?**
- **Respuesta: `let` respeta el alcance de bloque, `var` no**
