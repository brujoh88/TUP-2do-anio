| JavaScript (analogía casa) | Otorga interactividad y comportamiento (electricidad y mecanismos) |

## Preguntas y respuestas

**1. ¿Cuál es la función principal de JavaScript en el desarrollo web según la analogía de la casa?**
- **Respuesta: Otorgar interactividad y comportamiento (electricidad y mecanismos)**

**2. JavaScript es un lenguaje dinámicamente tipado. Esto significa que:**
- **Respuesta: El tipo de dato se determina automáticamente según el valor asignado**

**3. ¿Qué tipo de dato se obtiene cuando se declara una variable pero no se le asigna ningún valor?**
- **Respuesta: `undefined`**

**4. ¿Qué resultado devuelve la operación `10 % 3` en JavaScript?**
- **Respuesta: `1`**

**5. ¿Cuál es el resultado de la comparación `5 === "5"`?**
- **Respuesta: `false`**
